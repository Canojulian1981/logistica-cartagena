Configurar sincronización

Los datos se guardan directamente en tu repositorio de GitHub. Configura el acceso una sola vez por dispositivo — el navegador lo guarda de forma segura y no te lo vuelve a pedir.

1
Ve a github.com/settings/tokens/new, crea un token con permiso repo y cópialo aquí abajo.
2
Ingresa tu usuario de GitHub y el nombre del repositorio donde subiste esta app (ej: logistica-cartagena).
3
¡Listo! Los datos se guardan como un archivo datos.json en tu repositorio y se sincronizan en todos los dispositivos.
